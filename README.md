# 🚀 Nexus Inventory

Sistema Integral de Gestión Empresarial con Inventario Inteligente desarrollado como proyecto Full Stack.

## 📌 Descripción

Nexus Inventory es una aplicación web que permite administrar inventarios, productos, usuarios, ventas y reportes mediante una arquitectura basada en API REST.

El sistema incorpora autenticación por roles, dashboard administrativo y generación de reportes para optimizar la gestión empresarial.

## ⚙️ Instalación

### Clonar el repositorio

```bash
git clone https://github.com/mcerronvelasquez23/Nexus-Inventory.git
```

### Instalar dependencias

```bash
npm install
```

### Ejecutar el backend

```bash
cd api
npm install
npm start
```

### Ejecutar el frontend

```bash
cd client
npm install
ng serve
```
